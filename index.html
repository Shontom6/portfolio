Projects

Electrical Enclosure – Sheet Metal Design

This electrical enclosure was designed in SOLIDWORKS using Sheet Metal features. The model includes base panels, ventilation slots, mounting holes, and a front access door for component installation and maintenance. Features such as Base Flange, Edge Flange, Cut-Extrude, and Sheet Metal bends were used to create a manufacturable enclosure structure.

Electrical Enclosure – Exploded View

An exploded view of the electrical enclosure assembly created in SOLIDWORKS to illustrate the relationship and assembly order of the enclosure body, front panel, internal components, and fasteners. Explode lines were added to clearly show the positioning and assembly sequence of each part.

Weldment Chair – SOLIDWORKS

This chair frame was designed in SOLIDWORKS using the Weldment feature. A 3D sketch was created to define the structure, and solid round profiles were applied using the Structural Member tool to build the frame. Trim/Extend was used to refine the intersections and simulate welded joints. This project demonstrates weldment modeling and structural frame design.

Structural Support Frame

This structural frame was designed using the SOLIDWORKS Weldment feature. A 3D sketch was used to define the frame layout, and square structural members were applied using the Structural Member tool. Trim/Extend was used to clean intersections and prepare welded joints. This project demonstrates weldment modeling and structural frame design.

actuator mounting

Designed a clevis-style actuator mounting bracket using SolidWorks. The model includes two parallel mounting arms with pin holes for actuator connection and three base holes for structural mounting. Fillets and smooth radius transitions were applied to improve strength and manufacturability. A fully dimensioned 2D manufacturing drawing was created with standard views and material specified as Aluminum 6061.
